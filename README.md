
# SyncEthic AI SDK Template

Official React/Next.js template for the [SyncEthic AI SDK](https://syncethicai.com). Embed a powerful, intelligent chatbot into your web application in seconds.

## Features

- 🚀 **Next.js 15+** with App Router
- 🎨 **Tailwind CSS** for easy styling
- 🤖 **Official @syncethic/sdk** integration
- 📱 **Responsive Design** mobile-friendly out of the box
- ✨ **Animated Call-to-Action** to drive user engagement

### 1. Clone the repository

```bash
git clone https://github.com/your-username/syncethic-sdk-template.git
cd syncethic-sdk-template
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env.local` file in the root directory and add your SyncEthic API key:

```bash
NEXT_PUBLIC_SYNCETHIC_API_KEY=sk_global_your_api_key_here
NEXT_PUBLIC_SYNCETHIC_BOT_SLUG=finance_advisor
```

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
